# Nitech Map

## 概要
名古屋工業大学の学生の為のスマホアプリ.  
入力された時間割情報と現在時刻から、次の授業の場所を地図上で赤く強調します.  
アプリを開くだけで講義室を確認できます.  
<a href="https://play.google.com/store/apps/details?id=com.c0de.nitechmap_c0de">Google Play Store</a>  

## 使用技術
- Futter
- Dart
- Providerパッケージ
